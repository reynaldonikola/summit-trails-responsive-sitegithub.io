# Summit Trails – Responsive Web Experience

This project is a responsive website created for Project 4 (CSS Theme and Refactor).

## Description
The goal of this project was to improve a previous website by refactoring CSS, creating a new theme, adding a theme toggle, and making the site mobile responsive.

## Features
- Clean and organized CSS using Flexbox and Grid
- Fully responsive layout for mobile and desktop
- Separate theme stylesheet (dark mode)
- Toggle button to switch between themes
- Improved spacing, alignment, and visual hierarchy

## File Structure
project4/

## How It Works
- The default design loads from `main.css`
- When the button is clicked, `theme.css` is dynamically added
- The theme overrides the base styles without modifying them
